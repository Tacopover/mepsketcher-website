About MepSketcher

MepSketcher is a specialized CAD-like application designed specifically for MEP (Mechanical, Electrical, and Plumbing) professionals. Work directly with your PDF drawings to create, manage, and visualize complete MEP networks with precision and ease.

Whether you're designing HVAC systems, electrical networks, or plumbing layouts, MepSketcher provides the tools you need to streamline your workflow and improve project efficiency.

Getting Started

1

System Requirements

  • Windows 10 or later
  • .NET Framework 4.8
  • Minimum 4GB RAM (8GB recommended)
  • Screen resolution: 1280x720 or higher
2

Installation

  • Download the installer from the link below
  • Run the installer as Administrator
  • Follow the installation wizard
  • Launch MepSketcher from your Start Menu
3

Basic Workflow

  • Open your PDF drawing
  • Create a new building and level structure
  • Select network types and drawing tools
  • Start placing terminals, equipment, and segments
